# PowerBi-Dashboards

# 1. Project Title / Headline

# IPL & Mumbai Indians Analytics Dashboard

### An interactive and data-driven Power BI dashboard built to analyze IPL seasons from 2008–2025 with deep insights into Mumbai Indians’ team performance, batting trends, bowling efficiency, match strategies, and player analytics.

---

# 2. Short Description / Purpose

The Dashboard is a visually engaging and analytical Power BI project designed to explore complete IPL statistics while focusing specifically on Mumbai Indians (MI). The dashboard provides season-wise performance analysis, player comparisons, batting and bowling insights, scoring patterns, toss impact, and strategic match analysis using interactive visualizations and dynamic filters.

This dashboard is intended for cricket enthusiasts, sports analysts, IPL followers, data analytics learners, and recruiters who want to understand how data visualization can transform raw cricket data into meaningful insights and storytelling.

---

# 3. Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 [Microsoft Power BI Desktop](https://powerbi.microsoft.com?utm_source=chatgpt.com) – Main platform used for dashboard creation and data visualization.
* 📂 Power Query Editor – Used for data cleaning, transformation, null handling, and data preparation.
* 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, dynamic calculations, and custom analytics.
* 🔗 Data Modeling – Relationships established between IPL Matches data and Ball-by-Ball data for accurate cross-filtering and analysis.
* 📁 CSV Dataset Files – Used as the primary data source format.
* 📈 Interactive Visualizations – Built using bar charts, line charts, scatter plots, KPI cards, slicers, and tables.

# 5. Features / Highlights

## • Business Problem

IPL generates massive amounts of cricket data every season, but raw datasets are difficult to interpret without proper visualization and analysis.

Questions such as:

* Which seasons were Mumbai Indians’ strongest?
* Which batsmen contributed most consistently?
* How does MI perform while chasing vs defending?
* Which bowlers are most economical?
* During which overs does MI score aggressively?

…cannot be answered easily using spreadsheets or raw match data alone.

---

# • Goal of the Dashboard

To develop an interactive cricket analytics dashboard that:

* Provides complete IPL overview analysis.
* Delivers deep performance insights for Mumbai Indians.
* Enables users to analyze batting, bowling, and match strategies visually.
* Demonstrates practical skills in Power BI, DAX, and sports analytics.
* Converts raw cricket data into meaningful visual storytelling.

# • Walkthrough of Key Visuals

---

## 🏆 IPL Overview Section

### Key KPI Cards

Displays:

* Total IPL Seasons
* Total Matches
* Champions & Runner-ups
* Orange Cap Holder
* Purple Cap Holder

Helps users quickly understand overall IPL statistics.

---

## 📊 Dynamic Points Table

Interactive points table displaying:

* Wins
* Losses
* Net Run Rate
* Qualification status
* Top 4 and Top 6 teams

Allows season-wise team performance analysis.

---

## 💙 Mumbai Indians (MI) Performance KPIs

Displays:

* Total Runs
* Total Fours
* Total Sixes
* Total Matches
* Matches Won

Provides a quick overview of MI’s historical performance.

---

## 📈 Runs by Overs Analysis (Line Chart)

Visualizes how MI scores across different overs.

Overs are categorized into:

* Powerplay (1–6)
* Middle Overs (7–15)
* Death Overs (16–20)

Helps identify scoring acceleration patterns and batting strategy.

---

## 🏏 Top MI Batsmen Analysis (Bar Chart)

Ranks Mumbai Indians batsmen based on:

* Total Runs
* Batting contribution
* Consistency

Highlights the most impactful batting performers.

---

## 🎯 Strike Rate vs Runs Analysis (Scatter Plot)

Compares MI batsmen based on:

* Strike Rate
* Total Runs
* Batting Aggression

Helps identify explosive and consistent players.

---

## 🔥 Top MI Bowlers Analysis (Bar Chart)

Displays:

* Highest wicket-taking bowlers
* Bowling impact
* Bowling consistency

Highlights MI’s strongest bowling performers.

---

## 🎳 Economy Rate Analysis

Analyzes bowlers based on:

* Runs conceded
* Bowling efficiency
* Economy rate

Helps compare bowling control and performance effectiveness.

---

## 🧠 Toss Impact & Match Result Analysis

Compares:

* Toss won vs matches won
* Chasing vs defending performance
* Match-winning strategies

Provides strategic match insights.
